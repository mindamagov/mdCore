# mdCore

Core-плагин для Paper 1.16.5 (Очень полезно для Box-PvP)

## Функции

- Сообщения об убийствах в чате  
- Блокировка рекламы  
- Приветственное сообщение при входе  
- Ограничение постановки и ломания блоков  
- Автоматическое удаление временных блоков

## Требования

- Paper 1.16.5  
- Java 17

## Установка

1. Скачать jar-файл  
2. Поместить в папку plugins  
3. Перезапустить сервер  
4. Настроить config.yml в plugins/mdCore/

## Сборка из исходников

```bash
git clone https://github.com/ВАШ_НИК/mdCore.git
